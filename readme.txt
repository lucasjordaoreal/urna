==== COMO VOTAR ====

1. VOTANDO EM UM CANDIDATO:
- Clique nos números do teclado da urna correspondentes ao seu candidato (ex: 2 e 2 para Flávio Bolsonaro).
- A foto, o nome e o partido do candidato aparecerão na tela.
- Se os dados estiverem corretos, clique no botão verde "CONFIRMA".
- O som de confirmação tocará e aparecerá a mensagem "FIM" na tela.

2. CORRIGINDO UM ERRO:
- Se você digitou um número errado, basta clicar no botão laranja "CORRIGE".
- A tela será apagada e você poderá digitar o número correto novamente.

3. VOTANDO EM BRANCO:
- Para anular seu voto intencionalmente em branco, clique no botão branco "BRANCO".
- Aparecerá a mensagem "VOTO EM BRANCO" na tela.
- Em seguida, clique no botão verde "CONFIRMA" para registrar.

==== DEPENDÊNCIAS ====
Antes de executar o programa `urna.py`, lembre-se de ter o Pillow instalado.
Caso não tenha, rode o comando:
pip install -r requirements.txt
